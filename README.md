# dividend

# 개요 
배당금 정보를 제공하는 서비스 클론 프로젝트

Use : Spring, Jpa, H2, Redis, Security, Jwt

### 구현해야할 API 리스트
- [x] 회사 배당금 정보 반환 API
- [x] 회사 이름 자동완성 API
- [x] 회사 목록 반환 API
- [x] 회사 정보 추가 API
- [x] 회사 정보 삭제 API
- [ ] 회원가입 API
- [ ] 로그인 API

### 부가 기능
- [x]  yahoo finance 스크래핑

- [x]  통합 에러 처리
- [ ]  테스트 코드 작성
- [ ]  로그레벨로 필요한 로그 남기기
